HUDDLE MEETING GUIDE (For Manager):

Keep It Short - 5 Minutes Maximum!

1. Quick Welcome (30 seconds)
   - "Good morning team! Let's have quick meeting."
   - Take attendance (who's here?)

2. Review Yesterday's Incomplete Tasks (1 minute)
   - "These tasks not done yesterday. Who will finish today?"
   - Assign someone to each task

3. Today's Priorities (1 minute)
   - "Today most important: 1, 2, 3"
   - Make sure everyone understands

4. Team Assignments (1 minute)
   - "Here's what everyone does today"
   - Answer quick questions

5. Problems/Safety (1 minute)
   - "Any equipment broken? Any problems?"
   - "Safety reminder: [topic]"

6. Wins/Announcements (30 seconds)
   - "Good job yesterday to [name]!"
   - Quick announcements

7. Questions (30 seconds)
   - "Any questions? Ask now or come to me later."

Done! Get to work!

TIPS FOR MULTILINGUAL TEAMS:

Use Visual Cues:
- Check mark = Done/Complete
- X = Not done/Problem
- ! = Warning/Important
- Person icon = Person/Who
- Box = Order/Package
- Wrench = Equipment/Tool
- Clock = Time/Deadline

Simple English:
- Use short words
- Speak slowly
- Use pictures/gestures
- Write important things
- Repeat if needed

Common Phrases:
- "You do [task]" = pointing + task
- "Need help?" = gesture asking
- "Good job!" = thumbs up
- "Come here" = wave hand
- "Wait" = hand up
- "Go" = point direction

Translation Apps:
- Google Translate (phone app)
- Microsoft Translator
- iTranslate
- Have team member help translate
