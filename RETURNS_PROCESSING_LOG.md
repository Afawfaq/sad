## Return Reasons (Common)
- Wrong item ordered
- Defective/damaged
- Not as described
- Changed mind
- Duplicate order
- Better price elsewhere
- Late delivery

## Processing Guidelines

### Condition Assessment
- **New/Unopened:** Can resell as new
- **Used/Opened:** Inspect carefully, may resell as open box or return to vendor
- **Damaged:** Cannot resell, damage out or return to vendor

### Actions
- **Restock:** Return to warehouse location, update inventory
- **Damage Out:** Mark as damaged, remove from inventory, document for vendor claim
- **Dispose:** Discard if unsellable and no vendor return option
