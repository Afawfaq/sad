# Order & Shipping Issues

Fast solutions for order and shipment problems.

## üì¶ "Where is order #_____?"
**Quick Check:**
1. Check system - is it picked/packed/shipped?
2. Check pick queue - assigned to someone?
3. Check packing stations - sitting somewhere?
4. Check shipping dock - waiting for carrier pickup?

**If can't find:** Assign to your fastest picker, mark as priority

## üö® "Customer needs it TODAY but it's still here"
**Actions:**
1. Check time - Can we make last carrier pickup?
2. If yes: Pull it to front of line, your best packer, verify accuracy
3. If no: Inform customer service immediately (don't wait)
4. Consider: Local courier service (if approved by management)
5. Document: Why it wasn't shipped earlier

**Prevention:** Review picking process, retraining if needed

## üì¶ "Order is damaged"
**Actions:**
1. Check packing station - how was it packed?
2. If our fault: Immediate reship, note packer for coaching
3. If carrier damage: File carrier claim
4. Review: Does this item need better packaging?
