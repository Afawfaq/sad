COMMUNICATION GUIDELINES

When to Loop in Store Manager:
- Escalation affects store reputation
- Pull inventory from store floor
- Customer return online order to store
- Warehouse delay → store customer issue
- Need extra hands for warehouse

When Store Manager Loops You In:
- Store customer wants shipping
- Store receiving warehouse shipment
- Customer return/exchange online order in store
- Store inventory discrepancy
- Need warehouse staff in store

How to Communicate:
Urgent (15 min response): Text/call - Customer waiting, safety, crisis
Important (1 hour): Walkie/radio - Staffing, inventory, coordination
Normal (same day): Email/notes - Updates, planning, documentation

CONFLICT RESOLUTION

If You Disagree on Priority:
1. Both state case (facts)
2. Consider: Who has more urgent customer need?
3. Consider: Can we do both?
4. Decide: Use priority guide
5. Still stuck: Escalate together

Priority Guide:
1. Safety issue → Always first
2. Customer physically waiting → Usually priority
3. P1 order (ship 4 hours) → High priority
4. Store sale vs online → Equal
5. P2/P3 orders → Can wait

If You Disagree on Resources:
1. What's actual impact?
2. Can employee split time?
3. Alternative person/solution?
4. Which is more time-sensitive?
5. Compromise

Remember: Same team, same goals

TIPS FOR WORKING WELL TOGETHER

Do:
- Communicate early and often
- Respect each other's priorities
- Find win-win solutions
- Back each other up
- Share credit
- Give heads-up on problems
- Be flexible

Don't:
- Surprise each other
- Compete for resources
- Go over each other's head
- Blame the other area
- Hoard information
- Make decisions without input
- Let small issues grow
